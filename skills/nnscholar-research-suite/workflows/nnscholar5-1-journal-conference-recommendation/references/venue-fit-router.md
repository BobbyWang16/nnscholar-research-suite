# Venue Fit Router

Use this router to classify the manuscript before recommending venues.

## Fit Questions

1. What is the article type: original research, review, methods, case report, benchmark, materials performance, theoretical article, or conference paper?
2. What is the evidence strength: exploratory, confirmatory, externally validated, mechanistic, benchmarked, interpretive, or proposal-stage?
3. What is the main audience: specialist, clinical, computational, interdisciplinary, applied, policy, engineering, or humanities?
4. What constraints matter: review speed, fee, impact/rank, indexing, language, region, double-blind, open access, deadline?

## Discipline Signals

| Discipline | Must check |
|---|---|
| Clinical medicine | patient relevance, validation, ethics, reporting checklist, clinical endpoints |
| Basic biomedicine | mechanism, model system, controls, replicates, image/assay depth |
| AI/data science | baselines, benchmarks, ablation, reproducibility, venue deadline, anonymity |
| Materials/chemistry | novelty, characterization, performance comparison, mechanism, stability |
| Education/psychology | participant sample, instruments, reliability, ethics, intervention |
| Economics/social science | identification, data source, robustness, policy/theory contribution |
| Humanities | argument, sources/corpus, theoretical conversation, journal tradition |
| Engineering | system validation, workload, reproducibility, application relevance |
