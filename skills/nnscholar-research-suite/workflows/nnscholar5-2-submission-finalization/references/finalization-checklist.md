# Finalization Checklist

## Core Items

- Target venue and article type confirmed.
- Manuscript follows required section order.
- Abstract format and word count checked.
- Title page and blinded manuscript requirements checked.
- Figures/tables match count, format, resolution, caption, and file naming requirements.
- Supplementary files listed and named.
- References and citation style checked.
- Ethics, consent, funding, COI, author contributions, data/code availability checked.
- Reporting checklist selected when relevant.
- Terminology ledger is consistent across manuscript, figures, tables, and supplement.
- Central figures/tables have source-data or repository routes.
- Data/code statements match available files, repository identifiers, accessions, README/metadata, license, and any restricted-data procedure.
- Central claims and reused datasets have verified sources.

## Submission Blockers

- Missing ethics/consent statement.
- Missing required author declarations.
- Manuscript exceeds hard word limit.
- Required blinded files are not prepared.
- Figure/table files missing or low resolution.
- Required supplementary data/code not prepared.
- Central data lack durable access route, repository identifier, accession, DOI, or documented restricted-access process.
- Source data are missing for central figures/tables.
- Data/code availability statements conflict with the actual package.
