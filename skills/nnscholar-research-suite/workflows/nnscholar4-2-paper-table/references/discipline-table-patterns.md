# Discipline Table Patterns

## Clinical Medicine

Common tables:
- Table 1 baseline characteristics;
- model performance table;
- adverse event/toxicity table;
- subgroup or multivariable analysis table.

Required notes:
- data are n (%), mean (SD), or median (IQR);
- test used for p values;
- missing data handling;
- model adjustment covariates;
- abbreviation definitions.

## Basic Biomedicine

Common tables:
- reagent/antibody/primer table;
- assay condition table;
- differential expression/enrichment summary;
- biological replicate summary.

Required notes:
- biological vs technical replicates;
- normalization/control;
- assay condition and units.

## AI / Data Science

Common tables:
- benchmark table;
- ablation table;
- dataset statistics table;
- hyperparameter table.

Required notes:
- metric direction;
- train/validation/test split;
- seeds/folds/repetitions;
- mean/SD or CI;
- best/second-best formatting policy.

## Materials / Chemistry

Common tables:
- composition and synthesis condition table;
- physicochemical characterization table;
- performance comparison table;
- stability/cycling table.

Required notes:
- units;
- test condition;
- batch number/repetition;
- instrument settings if important.

## Education / Psychology

Common tables:
- participant baseline table;
- scale reliability/descriptive table;
- intervention outcome table;
- mediation/moderation model table.

Required notes:
- scale range and direction;
- reliability coefficients;
- time points;
- attrition/missingness.

## Economics / Social Science

Common tables:
- descriptive statistics;
- regression table;
- robustness check table;
- heterogeneity table.

Required notes:
- fixed effects;
- clustered/robust standard errors;
- controls;
- sample restriction;
- significance stars.

## Humanities

Common tables:
- corpus/source table;
- coding framework table;
- comparative case table;
- evidence map.

Required notes:
- corpus/source selection;
- coding rules;
- period/genre/language;
- interpretive limits.

## Engineering

Common tables:
- system specification table;
- experiment condition table;
- performance comparison table;
- failure/safety test table.

Required notes:
- hardware/software environment;
- workload/operating condition;
- units;
- repetitions and variance.
